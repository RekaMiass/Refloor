# Refloor — 3D-конфигуратор укладки пола

Браузерный инструмент: задаёте размеры комнаты и параметры ламелей, получаете интерактивную 3D-визуализацию и расчёт материалов.

## Стек

- React 19 + TypeScript
- Three.js / @react-three/fiber / @react-three/drei
- Vite

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173` в браузере.

### Сборка для продакшена

```bash
npm run build      # компиляция TypeScript + бандл Vite → папка dist/
npm run preview    # локальный предпросмотр собранного бандла
```

## Возможности

- Два режима укладки: **ёлочка** (herringbone) и **прямой** (straight)
- Слайдеры для настройки комнаты (длина 2–15 м, ширина 2–15 м, высота 2–4 м)
- Слайдеры для настройки ламели (длина 300–1200 мм, ширина 60–200 мм, зазор 0–5 мм)
- Выбор цвета стен из палитры или через color picker
- Вращение/зум 3D-сцены мышью (OrbitControls)
- Расчёт: площадь пола, площадь стен, расход краски, метраж плинтуса, количество досок

## Допущения в расчётах

| Параметр | Допущение |
|---|---|
| Комната | Прямоугольная, без дверей, окон и колонн |
| Потолок | В площадь покраски не включён |
| Плинтус | Периметр комнаты без вычета дверных проёмов: `2 × (длина + ширина)` |
| Покраска стен | 2 слоя; расход 1 л / 10 м² за слой; итог округляется до 0,5 л |
| Доски (прямой ряд) | Запас на подрезку: +10% к чистому количеству |
| Доски (ёлочка) | Дополнительные потери на ёлочку: +15%, затем ещё +10% на подрезку |
| Толщина доски | Фиксированная — 18 мм (только для 3D-визуализации) |
| Подложка | Не учитывается |
| Раскладка прямых рядов | Чётные ряды смещены на половину длины ламели (кирпичная перевязка) |

Расчёты — оценочные; для точного заказа материалов уточняйте потери у поставщика.
